# move-my-car
移车通知

## 目录说明

### fake-privacy-number
实现的是一个假的隐私号页面，用于模拟隐私号挪车码功能。

**注意：** 此页面并非真实的隐私号服务，仅用于展示和测试目的。页面中的电话号码为模拟号码，点击拨打按钮不会连接到真实的车主。

## fake-privacy-number 部署和使用说明

### 部署方式

1. **本地部署**
   - 将项目克隆到本地
   - 进入 `fake-privacy-number` 目录
   - 启动本地服务器（如 `python3 -m http.server 8000`）
   - 访问 `http://localhost:8000/generator.html` 开始使用

2. **服务器部署**
   - 直接部署 `fake-privacy-number` 目录下的所有文件（移除 fake-privacy-number 目录层级）
   - 确保服务器支持静态文件访问
   - 通过服务器域名或 IP 直接访问 `generator.html`
   - 请自行解决服务器，域名，SSL 证书问题

### 使用步骤

#### 1. 生成挪车码

1. 访问 `generator.html` 页面
2. 在输入框中填写真实的 11 位手机号码
3. 点击"一键生成"按钮
4. 系统会生成：
   - 压缩后的序列（seq 参数）
   - 完整的访问链接
   - 包含二维码和挪车提示的图片

#### 2. 保存和打印

1. 截图保存生成的二维码及挪车提示信息
2. 使用彩色打印，选择高质量打印模式
3. 确保二维码清晰可扫

#### 3. 张贴使用

1. 将打印好的挪车联系信息张贴到车辆适当位置
2. 建议张贴在副驾驶位前方挡风玻璃内侧下方
3. 确保不干扰驾驶视线，同时保证他人可视和方便扫码

#### 4. 他人使用

1. 他人通过扫描二维码访问 `index.html` 页面
2. 系统会自动抓取 URL 中的 seq 值并转换为手机号码
3. 点击"一键呼叫公主"按钮联系车主

### 功能说明

1. **手机号压缩**
   - 使用 62 进制算法压缩手机号码
   - 使用固定乱序字符集，确保生成的序列更加安全
   - 压缩后的序列作为 URL 参数传递

2. **链接生成**
   - 生成包含 seq 参数的完整访问链接
   - 链接格式：`https://[域名]/index.html?seq=[压缩序列]`

3. **二维码生成**
   - 自动生成包含访问链接的二维码
   - 二维码下方包含使用提示信息
   - 方便他人扫描使用

4. **seq 值解析**
   - `index.html` 页面会自动抓取 URL 中的 seq 参数
   - 使用相同的映射表将 seq 值转换回手机号码
   - 转换失败时使用默认手机号码作为 fallback

### 注意事项

1. **隐私保护**
   - 虽然此服务为模拟服务，但建议不要使用真实的重要手机号码进行测试
   - 生成的链接应妥善保管，避免被滥用

2. **技术限制**
   - 仅支持 11 位手机号码
   - 生成的链接在不同环境下访问时，需要确保域名或 IP 正确

3. **打印质量**
   - 建议使用高质量打印，确保二维码清晰可扫
   - 避免二维码被遮挡或污染

4. **定期检查**
   - 定期检查张贴的二维码是否清晰可扫
   - 如有损坏或模糊，请及时重新生成和张贴

### 示例使用流程

1. **车主操作**
   - 访问 `generator.html` 输入手机号 13888888888
   - 生成二维码并打印张贴

2. **他人操作**
   - 扫描二维码，访问 `index.html?seq=[压缩序列]`
   - 页面显示手机号 13888888888
   - 点击"一键呼叫公主"按钮尝试联系
